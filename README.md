## Todo App (TS React)
It's a awesome looking todo app. User can create or add his own list. And add todo task within his own list. User can delete his task after select the task. 

#### [Live](https://todo-app-tsr.netlify.app/)

## Technology
- React 
- TypeScript 
- SASS 
- JSX
